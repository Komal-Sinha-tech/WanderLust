# WanderLust 🌍

WanderLust is a full-stack travel listing web application where users can explore, create, edit, and review vacation stay listings.  
The project is inspired by platforms like Airbnb and is built using Node.js, Express.js, MongoDB, and EJS.

---

## 🚀 Features

- User Authentication & Authorization
- Create, Edit, and Delete Listings
- Upload Listing Images
- Add Reviews and Ratings
- Flash Messages & Form Validations
- Responsive UI
- Session & Cookie Authentication
- RESTful Routing
- MongoDB Database Integration

---
